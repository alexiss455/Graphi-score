# Graphi-score
Discover the Best Graphics Performance for Your Needs with Our Comprehensive GPU Rating Website. Explore the Latest GPUs, Compare Specifications, Performance Metrics, User Ratings, and Real-World Reviews. Make an Informed Decision with Ease - Your One-Stop Shop for GPU Selection.

# Features
- Browse through the list of GPUs available on the homepage.
- Click on a specific GPU to see its detailed page, including ratings and reviews.
- To leave a rating and review, click the "Write a Review" button and fill out the form.
- To filter reviews by criteria such as rating or date, use the filtering options provided.
- To submit a comment on a review, click the "Comment" button and add your thoughts.

- Getting Started
To use GraphiScore, simply visit https://graphiscore.cyclic.app/ in your web browser. No installation or setup is required.

- Back-end: Express, Node.js
- Database: MongoDB, Mongoose

The design of this application was inspired by the Cafely website, which can be found at https://cafely.vercel.app/. The clean and simple layout, neutral color palette, and use of high-quality images were elements that I sought to incorporate into this project.
